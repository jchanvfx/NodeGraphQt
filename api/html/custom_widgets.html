Built-in Widgets¶

The NodeGraphQt framework comes included with a few custom widgets.

Classes:

The NodeGraphQt.PropertiesBinWidget is a list widget for displaying and editing a nodes properties.

The NodeGraphQt.NodesPaletteWidget is a widget for displaying all registered nodes from the node graph in a grid layout with this widget a user can create nodes by dragging and dropping.

Properties Bin¶

class NodeGraphQt.PropertiesBinWidget(parent=None, node_graph=None)¶

The NodeGraphQt.PropertiesBinWidget is a list widget for displaying and editing a nodes properties.

1from NodeGraphQt import NodeGraph, PropertiesBinWidget
2
3# create node graph.
4graph = NodeGraph()
5
6# create properties bin widget.
7properties_bin = PropertiesBinWidget(parent=None, node_graph=graph)

Nodes Palette¶

class NodeGraphQt.NodesPaletteWidget(parent=None, node_graph=None)¶

The NodeGraphQt.NodesPaletteWidget is a widget for displaying all registered nodes from the node graph in a grid layout with this widget a user can create nodes by dragging and dropping.

Implemented on NodeGraphQt: v0.1.7
1from NodeGraphQt import NodeGraph, NodesPaletteWidget
2
3# create node graph.
4graph = NodeGraph()
5
6# create nodes palette widget.
7nodes_palette = NodesPaletteWidget(parent=None, node_graph=graph)
8nodes_palette.show()
